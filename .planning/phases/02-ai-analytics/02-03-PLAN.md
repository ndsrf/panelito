## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| invoke SSE stream → client state | Untrusted streamed JSON (panel_update) crosses into React/panel state |
| presence channel → all clients | ai_streaming broadcast controls input lock for every participant |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-10 | Tampering | panel_update payload corrupts UI | mitigate | PanelWidgetSchema.safeParse() gate in use-ai-stream before any panelStore update; invalid payloads silently discarded (AI-05); never feed raw stream JSON to Recharts |
| T-02-11 | Denial of Service | malformed multi-line SSE chunks crash the consumer | mitigate | Buffer + split on \n\n boundary with remainder retention (Pitfall 1); JSON.parse wrapped so a bad event is skipped, not fatal |
| T-02-12 | Spoofing | presence payload clobbering clears typing indicator | mitigate | Merged track() payload preserves typing + displayName (Pitfall 3); never replace the full payload |
| T-02-SC | Tampering | npm/pip/cargo installs | accept | No new packages; framer-motion/zustand/zod already installed + slopcheck-approved (RESEARCH) |
</threat_model>
