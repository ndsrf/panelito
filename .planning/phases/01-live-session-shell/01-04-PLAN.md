Hook signature (`apps/web/hooks/use-viewport.ts`):

- `useViewport(): void` — client-side. On mount: sets `document.documentElement.style.setProperty('--app-height', \`${window.innerHeight}px\`)` (idempotent; the inline script in `app/layout.tsx` already set it pre-paint — this is the React-side re-affirmation). Then attaches `window.visualViewport.addEventListener('resize', updateKeyboardHeight)` and `'scroll'` (Safari Visual Viewport requires both events). The handler computes `kh = appHeight - (window.visualViewport?.height ?? appHeight)` and sets `--keyboard-height: ${Math.max(0, kh)}px`. Cleanup on unmount removes the listeners.
