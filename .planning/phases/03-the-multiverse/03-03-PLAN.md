# Plan: The Branching Experience

Implement the user interface for creating forks from messages and visually representing the branched timeline in the chat stream.

## Requirements
- **BRANCH-01**: Fork from any historical message to create a new parallel branch.
- **BRANCH-02**: Unique color and readable label (visual representation).
- **AI-09**: Branch Auto-Labeling (trigger and feedback).

## User Decisions
- **D-10**: Parent messages ("the trunk") are visually dimmed and separated by a "Branch Point" divider.
- **D-15**: Participants can only fork from human messages.
- **D-16**: Isolation feedback label above the chat input field.
