# Plan: Multiverse Foundation (Database & API)

Establish the core branching engine on the backend. This includes the database schema for branches, the auto-labeling service, and updated API routes to support ancestry-aware message retrieval and AI context isolation.

## Requirements
- **BRANCH-01**: Any session participant can fork from any message in the conversation history.
- **BRANCH-02**: Each branch is assigned a unique color and a readable label.
- **BRANCH-04**: Branch AI context isolation (messages from sibling branches are excluded).
- **AI-09**: Branch Auto-Labeling via flash model.

## User Decisions
- **D-01/D-02**: Fixed sequence color assignment from a 10-color deep saturation palette.
- **D-05**: Auto-labeling via flash model for 2-3 word semantic labels.
- **D-09**: Support ancestry-based message retrieval.
